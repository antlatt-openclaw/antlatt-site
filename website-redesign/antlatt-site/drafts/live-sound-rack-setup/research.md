## Typical Live Sound Rack Components

### Core Components (Signal Flow Order):

```
INPUT STAGE                    PROCESSING                    OUTPUT STAGE
─────────────                  ──────────                    ────────────
Wireless receivers    →    EQ / Dynamics              →    Main outputs
DI boxes                   Compressors/Limiters            Monitor sends
Mic preamps                 Noise Gates                     Power amps
                           Crossovers
                           Effects processors
```

### 1. Power Distribution (Bottom of Rack)
- **Power Conditioner/Sequencer** - Furman, ART, Monoprice Stage Right
- Features: Surge protection, voltage regulation, sequenced power-on
- Price range: $100-600
- *Best practice: Power goes at BOTTOM of rack - heaviest, and prevents cable runs over gear*

### 2. Wireless Microphone Systems
- **Popular brands:** Shure (BLX, ULX, SLX), Sennheiser (EW series)
- Rack-mount receivers (half-rack or full-rack units)
- Considerations:
  - Frequency coordination (use Shure or Sennheiser planning apps)
  - Antenna placement (front of rack or remote with paddles)
  - Battery monitoring from receiver display
- Price range: $300-3000+ per channel

### 3. Signal Processing
- **EQ:** 31-band graphic EQ for room tuning ($200-500)
- **Dynamics:** Compressors, gates, limiters
- **Effects:** Reverb, delay units (if not done digitally)
- **Crossovers:** For bi-amped or tri-amped systems

### 4. Mixers (Digital or Analog)
- **Rack-mount digital mixers:** Behringer XR series, Allen & Heath Qu-SB
- **Laptop integration:** DAW or mixing software (Reaper, Ableton, Pro Tools)
- Audio interface for laptop: Focusrite, Universal Audio, PreSonus

### 5. Guitar Rack Integration
- **Amp modelers:** Line 6 Helix, Fractal Axe-FX, Kemper
- **Rack-mounted preamps:** Marshall JMP-1, Mesa TriAxis
- **Power amps:** Tube or solid-state rack amps
- **Guitar wireless:** Shure GLX-D, Line 6 Relay

## Road Case Recommendations & Sizing

### Case Types:

| Type | Use Case | Price Range |
|------|----------|-------------|
| **Soft Rack Bag** | Light local gigs, <4U | $50-150 |
| **Roto-Molded (SKB)** | Regular gigging, durable | $150-400 |
| ** ATA Flight Case** | Touring, air travel | $300-800 |
| **Shock Mount Case** | Delicate gear, touring | $400-1000+ |
| **Rolling Road Case** | Heavy gear, mobile | $200-600 |

### Sizing Guide:

| Rack Units (U) | Typical Use |
|----------------|-------------|
| 2-4U | Power conditioner + small wireless |
| 6U | Processing + wireless + small mixer |
| 8-10U | Full live rig with power + processing |
| 12-16U | Full PA system brain + amps |
| 20U+ | Major touring rig |

### Anthony's Setup Observations:
- Appears to use **rolling cases** - best for mobile DJs, solo artists, small venues
- **Depth matters** - full-depth cases for amps, shallow for effects
- **Shock mount** recommended if frequently transported in vehicle

### Recommended Brands:
- **Budget:** Gator, On-Stage
- **Mid-range:** SKB, Odyssey
- **Professional:** Pelican, Anvil, Custom cases (RoadCases.com)

## Signal Flow for Live Sound

### Standard Signal Chain:

```
SOURCE               PROCESSING               OUTPUT
───────              ──────────               ──────
Microphones    →     Mixer Channels     →    Main L/R
DI boxes             (gain, EQ, comp)        Monitors
Wireless mics        Effects sends           Subwoofer feed
Instruments          Subgroups               Recording out
                     Main mix                Livestream
```

### Typical Rack Signal Flow:
1. **Wireless receivers** → Mixer inputs
2. **DI inputs** → Mixer inputs
3. **Mixer outputs** → Graphic EQ (room tuning)
4. **EQ outputs** → Crossover (if bi-amping)
5. **Crossover outputs** → Power amps
6. **Power amps** → Speakers

### Laptop Integration (Anthony's Setup):
1. **Audio interface** → Laptop (USB/Thunderbolt)
2. **Laptop** → DAW for:
   - Virtual soundcheck (recorded tracks)
   - Playback tracks/click tracks
   - Recording the show
   - Plugin processing
3. **Interface outputs** → Main mix or monitors

## Power Management

### Best Practices:
1. **Single power inlet** - Avoid ground loops
2. **Power sequencing** - Turn on in order (source → processing → amps)
3. **Separate circuits** - Lights and audio on different circuits if possible
4. **UPS consideration** - For laptop and digital mixer protection
5. **Voltage monitoring** - Furman units with voltage displays

### Power Budgeting:
| Device | Typical Draw |
|--------|--------------|
| Digital mixer | 50-150W |
| Wireless receivers | 10-30W each |
| Laptop + interface | 50-100W |
| Power amp | 200-2000W (varies wildly) |
| Effects processors | 20-50W each |

## Portability Considerations

### Design Principles:
- **"One trip from car"** goal - Everything in one rolling case
- **Patch panel** - All connections at rear, pre-wired
- **Laminated patch map** - Quick reference in case lid
- **Cable loom** - Pre-made bundles for common connections
- **Rolling case** - Wheels and handle essential

### Weight Management:
- Power amps at **bottom** (heaviest)
- Mixers/processors at **working height**
- Wireless at **top** (antenna access)
- **6U max** per case for solo handling (per Gearspace recommendation)

### Transport Protection:
- **Shock mount cases** for amp-heavy racks
- **Foam padding** between units
- **Rack screws** on all units (not just sitting there)
- **Lid storage** for cables and accessories

## Venue Integration

### Typical Connections Needed:
- **Main outputs**: XLR to house PA
- **Monitor sends**: XLR to wedge/IEM transmitters
- **Recording out**: USB/FireWire to laptop or dedicated recorder
- **Network**: Ethernet for digital snake or remote control
- **Power**: Single Edison or L6-20 connection

### Venue Communication:
- **Input list**: What inputs you're providing
- **Stage plot**: Where everything goes
- **Tech advance**: Pre-show communication about needs
